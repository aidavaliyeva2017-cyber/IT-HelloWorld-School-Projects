## How to Run a Project

1. Clone this repository:
```bash
   git clone https://github.com/aidavaliyeva2017-cyber/IT-HelloWorld-School-Projects.git
```
2. Navigate to the project folder:
```bash
   cd shooter-game
```
3. Install dependencies:
```bash
   pip install arcade
```
4. Run the game:
```bash
   python main.py
```
